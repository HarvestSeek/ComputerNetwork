    1. 二叉搜索树及其性质

    (1). 定义

     由一棵二叉树组织，每个结点包含属性 key、left、right、parent；对任何结点 x，其左子树的关键字不大于 x.key，右子树的关键字不小于 x.key。
     结点的前驱 (predecessor)、后继 (successor)：中序的前一个结点和后一个结点。

    typedef struct node {
    int key;
    struct node *left;
    struct node *right;
    struct node *parent;
} Node;

typedef struct Tree {
    Node *root;
} BST;

    (2). 性质

    ① 满足二叉树的所有性质。
    ② 在 BST 中，如果一个结点有两个孩子，那么它的前驱没有右孩子，后继没有左孩子。
     - 前驱为左子树中的最大元，它不可能再有右孩子，后继同理。
    ③ 在所有关键字互不相同的 BST 中，如果一个结点 x 的右子树为空，并且它存在一个后继 y，那么 y 是 { nodes | node 和 node.left 都是 x 的祖先 } 中的层数最低的元素（规定每个结点都是它自己的祖先）。
     - 首先，y 一定是 x 的祖先；然后，y.left 也一定是 x 的祖先，否则 x 会出现在 y 的右子树中；最后，假设 y 不是最低的，那么最低的那个一定出现在 y 的左子树中。

    Node* IterativeBSTSearch(Node *r, int key) {
    while (r != NULL && r->key != key) {
        if (r->key > key) {
            r = r->left;
        } else {
            r = r->right;
        }
    }
    return r;
}
    $O(h)$

    Node* Minimum(Node *r) {
    while (r->left != NULL) {
        r = r->left;

    return r;

Node* Maximum(Node *r) {
    while (r->right != NULL) {
        r = r->right;
    }
    return r;
}
    $O(h)$

    Node* Predecessor(Node *node) {
    if (node->left != NULL) {
        return Maximum(node->left);
    }
    Node *t = node->parent;
    while(t != NULL && t->left == node) {
        node = t;
        t = t->parent;
    }
    return t; // NULL: no pred
} 

Node* Successor(Node *node) {
    if(node->right != NULL) {
        return Minimum(node->right);
    }
    Node *t = node->parent;
    while(t != NULL && t->right == node) {
        node = t;
        t = t->parent;
    }
    return t;
}
    $O(h)$

    void Insert(BST *tree, Node *r, Node *iNode) {
    int key = iNode->key;
    Node *last;
    while (r != NULL) {
        last = r;
        if (key < r->key)  {
            r = r->left; 
        } else {
            r = r->right;
        }
    }
    iNode->parent = last;
    if (last == NULL) { 
        tree->root = iNode;
    } else if (key < last->key) {
        last->left = iNode;
    } else {
        last->right = iNode;
    }
}
    $O(h)$

    (4). DELETE

     设要删除的结点为 z，其后继为 y。删除总体分为三种情况，特殊情况是当 z 有两个孩子，这时选取它的后继 y 替换它，因为 y 一定没有左子树。更详细的过程为：

    • 如果 z 没有左孩子，那么用它的右孩子替换它。
    • 如果 z 有左孩子且只有一个孩子，用左孩子替换它。
    • 如果 z 有两个孩子，需要将 y 移出并拼接：
      • 如果 y 是 z 的右孩子，用 y 替换 z（并把 z 的左子树转接到 y 上）。
      • 否则，先用 y 的右孩子替换 y，然后再用 y 替换 z（并把 z 的左右子树都转接到 y 上）。

     提取一个子过程 TRANSPLANT：用一棵子树替换另外一棵子树。

    void Transplant(BST *tree, Node *des, Node *src) {
    if (des->parent == NULL) {
        tree->root = src;
    } else if (des->parent->left == des) {
        des->parent->left = src;
    } else {
        des->parent->right = src;
    }
    if (src != NULL) {
        src->parent = des->parent;
    }
} 

void Delete(BST *tree, Node *z) {
    if (z->left == NULL) {
        Transplant(tree, z, z->right);
    } else if (z->right == NULL) {
        Transplant(tree, z, z->left);
    } else {
        Node *y = Minimum(z->right); // succ
        if (y != z->right) {
            Transplant(tree, y, y->right);
            y->right = z->right;
            y->right->parent = y;
        }
        Transplant(tree, z, y);
        y->left = z->left;
        y->left->parent = y;
    }
}
    $O(h)$
