    2.3

    1.堆的定义

    (1). 二叉堆：一个用数组表示的完全二叉树(complete binary tree)
    (2). 最大堆：除了根结点以外的所有结点 $i$ 都满足 $A[PARENT(i)] \geq A[i]$
    (3). 最小堆：除了根结点以外的所有结点 $i$ 都满足 $A[PARENT(i)] \leq A[i]$
    (4). 结点的高度：该结点到叶结点最长简单路径上边的数目
    (5). 堆的高度：根结点的高度

    2.基本操作

    ① MAX-HEAPIFY(A, i):
     假设当前堆以 LEFT(i) 和 RIGHT(i) 为根结点的二叉树都是最大堆， A[i] 与这两个结点的关系不确定，该操作用于维护最大堆的性质。

    void MaxHeapify(Heap* H, int i) {
    int father = i;
    int child = 2*i;
    int root = H->A[i];
    while (child <= H->size) {
        if (child < H->size && H->A[child] < H->A[child + 1]) {
            child++;
        }
        if (H->A[child] < root) {
            break;
        }
        H->A[father] = H->A[child];
        father = child;
        child *= 2;
    }
    H->A[father] = root;
}
    $T(n)=O(lgn)=O(h)$

    void BuildMaxHeap(Heap* H) {
    int i;
    for (i = H->size / 2; i >= 1; i--) {
        MaxHeapify(H, i);
    }   
}
    自底向上，迭代通过 MaxHeapify(A, i) 调整最大堆。
    $T(n)=O(n)$，最大调整次数为各结点高度和。

    ③ HEAP-INCREASE-KEY(A, i, key):
     增大堆中某个元素的值。因为增大 A[i] 后，最大堆的性质可能被违背，在其到根结点的路径上不断与其父结点比较并交换，寻找适当的插入位置。

    void HeapIncreaseKey(Heap *H, int i, int key) {
    if (key < H->A[i]) {
        printf("error: new key is smaller");
    }
    while (i > 1 && H->A[i / 2] < key) {
        H->A[i] = H->A[i / 2];
        i = i / 2;
    }
    H->A[i] = key;
}
    $T(n)=O(lgn)$

    3.堆排序

    先建堆，随后不断把堆顶元素交换到末尾、缩小堆并调整，直到排好序。

    void HeapSort(Heap *H) {
    BuildMaxHeap(H);
    int i;
    for (i = H->size; i >= 2; i--) {
        swap(&H->A[i], &H->A[1]);
        H->size--;
        MaxHeapify(H, 1);
    }
}
    相比于快速排序，堆排序无论好坏情况，复杂度都是 $O(nlgn)$，但是它的复杂度中的常数较大，因此通常比快排慢。

    堆排序的特点：
    ① 堆排序不稳定
    ② 对缓存不友好
    ③ 无所谓最好、最坏、平均，运行时间为 $O(nlgn)$ (当所有元素相同时为 $O(n)$)

    4.优先队列(priority queue)

    一个最大优先队列应有以下操作：
     ① INSERT(S, x): 把元素 x 插入集合 S。
     ② MAXIMUM(S): 返回 S 中具有最大关键字的元素。
     ③ EXTRACT-MAX(S): 去掉并返回 S 中最大关键字元素。
     ④ INCREASE-KEY(s, x, k): 把 x 的值增加到 k。

    H->size++;

    int HeapExtractMax(Heap *H) {
    if (H->size < 1) {
        printf("heap underflow");
    }
    int max = H->A[1];
    H->A[1] = H->A[H->size];
    H->size--;
    MaxHeapify(H, 1);
    return max; 
}
