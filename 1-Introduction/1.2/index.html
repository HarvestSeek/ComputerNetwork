    1.2

    1.增长的阶

    $ \Theta(1) < \Theta(lgn) < \Theta(\sqrt n) < \Theta(n) < \Theta(nlgn) < \Theta(n^2) < \Theta(n^3) < \Theta(2^n) < \Theta(n!) $

    2.同阶函数集合

    $ \Theta(g(n)) = \lbrace \ f(n) \ | \ \exists c_1 \text{、} c_2 > 0, \ n_0 ,\ \forall n \geq n_0, \ \ c_1g(n) \leq f(n) \leq c_2g(n) \rbrace $

    如果$f(n) \in \Theta(g(n)), \ g(n) $与$f(n)$同阶，记作$ f(n) = \Theta (g(n))$

    3.低阶函数集合

    $O(g(n))=\lbrace \ f(n)\ \mid \ \exists c > 0, \ n_0, \ \forall n \geq n_0, \ 0 \leq f(n) \leq cg(n)\rbrace$

    4.高阶函数集合

    $\Omega(g(n))=\lbrace \ f(n) \ \mid \ \exists c > 0, \ n_0, \ \forall n \geq n_0, \ 0 \leq cg(n) \leq f(n) \rbrace$

    5.严格低阶

    $o(g(n))=\lbrace \ f(n)\ \mid \ \forall c > 0, \ n_0, \ \forall n \geq n_0, \ 0 \leq f(n) < cg(n)\rbrace$

    6.注

    ① $f(n)=\Theta(g(n)) \Rightarrow f(n)=O(g(n))$，Θ 标记强于 O 标记
    ② big O 标记可能不是紧的，例如 $2n^2=O(n^2)$是紧的，但 $2n=O(n^2)$不是
    ③ small o 标记用于标记上界但不是紧的的情况，例如 $2n^2 \ne o(n^2)$
